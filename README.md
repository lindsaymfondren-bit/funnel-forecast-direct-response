# Funnel Forecast Calculator

A lightweight forecasting tool for modeling funnel performance from top-of-funnel traffic through conversion and revenue outcomes.

## Overview
This calculator helps estimate what a funnel could produce based on a few key assumptions, such as traffic, conversion rates, and average value per conversion.

It is useful for quick scenario planning, rough forecasting, and testing how changes in one stage of the funnel affect the overall outcome.

## Features
- Input core funnel assumptions
- Estimate leads and customers
- Project revenue or return
- Compare different scenarios
- Reduce manual spreadsheet math

## Typical inputs
- Traffic or impressions
- Click-through rate
- Landing page conversion rate
- Lead qualification or close rate
- Average order value / contract value

## Typical outputs
- Leads generated
- Qualified leads
- Customers won
- Estimated revenue

## Use case
This project was created as a simple way to pressure-test funnel assumptions and make quick forecasting easier.
